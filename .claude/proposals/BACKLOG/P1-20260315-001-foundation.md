## Algorithm ABC Contract

```python
class Algorithm(ABC, Generic[TInput, TOutput]):
    @property
    @abstractmethod
    def proof_reference(self) -> str: ...

    @property
    @abstractmethod
    def complexity(self) -> str: ...

    @property
    @abstractmethod
    def error_bound(self) -> str: ...

    @abstractmethod
    def execute(self, input_data: TInput) -> TOutput: ...
```

Every concrete algorithm class's `__init__` parameters ARE the generation knobs.

## Configuration Architecture (3 layers + Jinja2 resolution)

**Resolution chain:**
```
docs/constants.md          → base namespace (PHI, GOLDEN_ANGLE, Gauss nodes, etc.)
    ↓
docs/proofs/*.md           → algorithm tunables, can reference {{constants.*}}
    ↓
docs/styles/*.md           → style config, can reference {{constants.*}} and {{proofs.*}}
    ↓
configs/*.yaml             → generation config, can override any value from above
```

## Constants Document Format (`docs/constants.md`)

```yaml
---

  PHI:
    value: 1.6180339887498949
```

## Proof Document Format

```yaml
---
type: proof
algorithm: GaussQuadratureArcLength
class_path: glyph_animator.algorithms.arc_length.GaussQuadratureArcLength
uses_constants: [GAUSS_5PT, PHI]
parameters:
  quadrature_order:
    value: 5
    type: int
    options: [3, 5, 7, 10]
    description: "Number of sample points"
  tolerance:
    value: 1.0e-12
    type: float
    range: [1.0e-15, 1.0e-6]
complexity: "O(1) per segment"
error_bound: "< 10⁻⁶ absolute for font-scale coordinates"
gate_validation: "Gate 3: measured error < 10⁻⁶ across all 10 digits"
---
```
